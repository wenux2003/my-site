
# How to Run Locally

## Prerequisites
- [Node.js](https://nodejs.org/) (v14 or higher)
- [Git](https://git-scm.com/)

## Installation Steps

1. **Clone the repository**
    ```bash
    git clone <repository-url>
    cd test-codes
    ```

2. **Install dependencies**
    ```bash
    npm install
    ```

3. **Run the code**
    ```bash
    npm start
    ```

## Troubleshooting

If you encounter issues:
- Ensure Node.js and npm are installed: `node --version` and `npm --version`
- Clear the node_modules cache: `rm -rf node_modules && npm install`
- Check that all environment variables are configured (if applicable)
